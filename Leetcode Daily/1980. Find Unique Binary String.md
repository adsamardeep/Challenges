## 🏷 Find a Unique Binary String Using Diagonal Flip (Cantor’s Diagonalization)

## 📜 Problem Statement
You are given `n` unique binary strings, each of length `n`. The task is to generate a binary string of length `n` that **does not exist in the given list**. The constructed string must differ from every string in the list in **at least one position**, guaranteeing uniqueness.

## 🔍 Example Input/Output

**Example**

Input:
```

nums = ["01", "10"]

```

Output:
```

"11"

````

Explanation:  
The generated string differs from both `"01"` and `"10"` at least at one position.

## 🧠 Approach

1. Since there are `n` strings of length `n`, we can use **Cantor’s Diagonalization technique** to construct a unique string.
2. Look at the diagonal elements of the list: `nums[i][i]`.
3. Flip each diagonal bit:
   - `'0' → '1'`
   - `'1' → '0'`
4. This guarantees the new string differs from the `i-th` string at position `i`.
5. Because it differs from every string in the list at least once, the constructed string **cannot be present in the list**.
6. Join all flipped bits to form the final result.

## 💻 Code 

```python
class Solution:
    def findDifferentBinaryString(self, nums: list[str]) -> str:
        
        flip = lambda x: '0' if x=='1' else '1'
        return ''.join(map(flip,[nums[i][i] for i in range(len(nums))]))
````

## ⏱ Complexity Analysis

**Time Complexity:** `O(n)`

* We iterate once through `n` elements to construct the result.

**Space Complexity:** `O(n)`

* The resulting binary string of length `n` is stored.

## 🧪 Edge Cases

1. **Single element**

   ```
   nums = ["0"]
   ```

   Output → `"1"`

2. **All zeros**

   ```
   nums = ["000","001","010"]
   ```

   The diagonal flip still produces a unique string.

3. **All ones**

   ```
   nums = ["111","110","101"]
   ```

4. **Large input size**
   The algorithm remains efficient because it performs a single pass.

5. **Almost complete set of combinations**
   Even if only one binary string is missing, this method will always produce a valid unique string.
