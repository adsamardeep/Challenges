🏷 Title: Find a Unique Binary String Using Diagonal Flip (Cantor’s Diagonalization)

📜 Problem Statement  
Given a list of `n` unique binary strings, each of length `n`, the goal is to construct a binary string of length `n` that **does not exist in the given list**. The solution must guarantee that the generated string differs from every string in the list in at least one position.

🔍 Example Input/Output  

Example Input:
nums = ["01","10"]

Example Output:
"11"

Explanation:
- The constructed string differs from both "01" and "10".
- Any binary string not in the list is valid.

🧠 Approach  

1. The problem guarantees that there are `n` binary strings each of length `n`.
2. Use **Cantor’s Diagonalization technique** to construct a new string.
3. For each index `i`, take the `i-th` character from the `i-th` string (`nums[i][i]`).
4. Flip that bit:
   - `'0' → '1'`
   - `'1' → '0'`
5. This guarantees the resulting string differs from the `i-th` string at position `i`.
6. Since it differs from every string at least once, it **cannot match any string in the list**.
7. Join all flipped bits to form the final binary string.

💻 Code 

```python
class Solution:
    def findDifferentBinaryString(self, nums: list[str]) -> str:
        
        flip = lambda x: '0' if x=='1' else '1'
        return ''.join(map(flip,[nums[i][i] for i in range(len(nums))]))
````

⏱ Complexity Analysis

Time Complexity: **O(n)**

* We iterate once through `n` elements to read and flip the diagonal bits.

Space Complexity: **O(n)**

* The resulting binary string of length `n` is constructed.

🧪 Edge Cases

1. **Single element case**
   nums = ["0"] → Output: "1"

2. **All zeros**
   nums = ["000","001","010"]
   The generated diagonal flip will still produce a unique string.

3. **All ones**
   nums = ["111","110","101"]
   Flipping ensures the constructed string differs at diagonal positions.

4. **Large input size**
   Since the algorithm runs in linear time, it scales efficiently for large `n`.

5. **Strings already diverse**
   Even if all binary combinations except one exist, the diagonal flip method guarantees finding the missing one.
