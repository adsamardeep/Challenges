# 🏷 Maximum Distance Between Valid Pairs

## 📜 Problem Statement  
This function finds the maximum distance `j - i` such that `i <= j` and `nums1[i] <= nums2[j]`. Both arrays are non-increasing (sorted in descending order), and the goal is to efficiently find the largest valid pair distance.

## 🔍 Example Input/Output  
Input:  
nums1 = [30, 29, 19, 5]  
nums2 = [25, 25, 25, 25, 25]  

Output:  
2  

Explanation:  
Valid pairs include:  
- (i=2, j=4): 19 ≤ 25 → distance = 2  
Maximum distance = 2  

## 🧠 Approach  
1. Use two pointers `i` and `j`, both starting at 0.  
2. Traverse both arrays simultaneously:
   - If `i > j`, increment `j` to maintain valid condition `i <= j`.  
3. For each pair:
   - If `nums1[i] <= nums2[j]`, it's a valid pair:
     - Update `ans = max(ans, j - i)`
     - Move `j` forward to try increasing distance  
   - Otherwise, increment `i` to find a smaller value in `nums1`  
4. Continue until either pointer reaches the end of its array.  
5. Return the maximum distance found.

## 💻 Code 
```python
class Solution:
    def maxDistance(self, nums1: List[int], nums2: List[int]) -> int:
        i = 0
        j = 0
        ans = 0

        while i < len(nums1) and j < len(nums2):

            if i > j:
                j += 1
                continue

            if nums1[i] <= nums2[j]:
                ans = max(ans, j - i)
                j += 1
            else:
                i += 1

        return ans
````

## ⏱ Complexity Analysis

Time Complexity: O(n + m), where `n` and `m` are lengths of `nums1` and `nums2`.
Space Complexity: O(1), constant extra space.

## 🧪 Edge Cases

* No valid pairs → returns 0.
* Single element arrays → checks only one pair.
* All elements valid → maximum distance is `len(nums2) - 1`.
* `nums1` always larger than `nums2` → no valid pair, result = 0.
* Different array lengths → handled naturally by two-pointer traversal.
