# 🔎 Check If Binary String Has Only One Segment of Ones

## 📜 Problem Statement

Given a binary string `s`, determine whether it contains **at most one contiguous segment of `'1'`s**.

In other words:
- All `'1'` characters must appear **together in one block**
- After a `'0'` appears, **no more `'1'` should appear**

Return **True** if the condition is satisfied, otherwise **False**.

---

## 🔍 Example Input / Output

### Example 1
**Input**
```

s = "1001"

```

**Explanation**

Two segments of `'1'` exist:

```

1 000 1

```

**Output**
```

False

```

---

### Example 2
**Input**
```

s = "110"

```

**Explanation**

Only one segment of `'1'`:

```

11 0

```

**Output**
```

True

```

---

### Example 3
**Input**
```

s = "111111"

```

**Output**
```

True

```

---

## 🧠 Approach (Step-by-Step)

1. Create a boolean variable `seen_zero` to track whether we have encountered a `'0'`.
2. Traverse the string character by character.
3. If the current character is `'0'`, mark `seen_zero = True`.
4. If the current character is `'1'` **after a zero has already been seen**, it means another segment of `'1'` has started.
5. In that case, return **False** immediately.
6. If the loop finishes without finding such a case, return **True**.

Essential idea:

```

1...1 0...0  (valid)
1...1 0...0 1...1  (invalid)

````

---

## 💻 Code 

```python
class Solution:
    def checkOnesSegment(self, s: str) -> bool:
        seen_zero = False

        for c in s:
            if c == '0':
                seen_zero = True
            elif seen_zero:
                return False

        return True
````

---

## ⏱ Complexity Analysis

**Time Complexity:**

```
O(n)
```

We scan the string once.

**Space Complexity:**

```
O(1)
```

Only one boolean variable is used.

---

## 🧪 Edge Cases

1. `"11111"` → True (only one segment)
2. `"00000"` → True (no ones)
3. `"10"` → True
4. `"101"` → False
5. `"110011"` → False

---

✅ **Key Insight:**
Once a `'0'` appears, seeing a `'1'` again means a **second segment of ones**, which violates the condition.
