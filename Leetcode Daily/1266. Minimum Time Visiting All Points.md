## ğŸ§  Approach

* The optimal movement between two points uses **diagonal moves as much as possible**.
* Each diagonal move reduces both x and y distance by 1.
* The minimum time between two points is therefore:

  ```
  max(|x2 - x1|, |y2 - y1|)
  ```

### Steps:

1. Iterate through consecutive point pairs.
2. For each pair, compute the Chebyshev distance.
3. Accumulate the total time.

## ğŸ’» Code

```python
from typing import List

class Solution:
    def minTimeToVisitAllPoints(self, points: List[List[int]]) -> int:
        ans = 0
        for i in range(1, len(points)):
            ans += max(
                abs(points[i][0] - points[i - 1][0]),
                abs(points[i][1] - points[i - 1][1])
            )
        return ans
```
