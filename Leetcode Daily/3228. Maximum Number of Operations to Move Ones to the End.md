Given a binary string `s`, you can perform an operation every time a `'0'` immediately follows one or more `'1'`s.  
Each such `'0'` contributes an amount equal to the total number of `'1'`s seen so far.  
Return the total number of such operations across the string.

# ğŸ” Example Input/Output


Explanation:  
- At index 2 (`s[2] = 0`), preceded by `1`, accumulated ones = 2 â†’ add 2.  
- At index 4 (`s[4] = 0`), preceded by `1`, accumulated ones = 3 â†’ add 3.  
Total = 2 + 3 = 5.

# ğŸ§  Approach
1. Maintain a counter `ones` for how many `'1'`s have been seen so far.  
2. Traverse the string character by character:
   - If the character is `'1'`, increment `ones`.
   - If the character is `'0'` **and** the previous character is `'1'`, it marks a valid 1â†’0 boundary â€” add `ones` to the result.
3. Return the accumulated `res` after the full traversal.

# ğŸ’» Code 
```python
class Solution:
    def maxOperations(self, s: str) -> int:
        ones = 0
        res = 0
        for i, c in enumerate(s):
            if c == '1':
                ones += 1
            elif i > 0 and s[i - 1] == '1':
                res += ones
        return res
````

* **Time Complexity:** `O(n)` â€” single pass over the string.
* **Space Complexity:** `O(1)` â€” constant extra memory.

# ğŸ§ª Edge Cases

* `s = "1010"`: correctly counts two transitions.
* `s` ending in `'1'`: handled naturally; only `'0'` after `'1'` matters.
