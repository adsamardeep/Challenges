- The number of **distinct even values** equals the number of **distinct odd values** in that subarray.

## ğŸ§  Approach

This solution uses a **brute-force sliding start with pruning** and a **clever visitation trick**:

### Key ideas:

1. Fix a starting index `i`
2. Expand the subarray to the right (`j`)
3. Track:

   * `A[0]` â†’ count of distinct even numbers
   * `A[1]` â†’ count of distinct odd numbers
4. Use a `seen` array with **index stamping**:

   * `seen[val] == i + 1` means value already counted for this start index
   * Avoids clearing the `seen` array every iteration
5. If `A[0] == A[1]`, update the answer
6. Early break when remaining length cannot beat current result

This keeps the solution efficient despite the nested loops.

---

## ğŸ’» Code 

```python
class Solution:
    def longestBalanced(self, nums: List[int]) -> int:
        n = len(nums)
        res = 0

        seen = [0] * (max(nums) + 1)

        for i in range(n):
            if n - i <= res:
                break

            A = [0, 0]  # A[0] = distinct evens, A[1] = distinct odds
            
            for j in range(i, n):
                val = nums[j]
                if seen[val] != i + 1:
                    seen[val] = i + 1
                    A[val & 1] += 1

                if A[0] == A[1]:
                    res = max(res, j - i + 1)

        return res
```
