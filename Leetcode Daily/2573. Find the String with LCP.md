## 🏷 Reconstruct String from Longest Common Prefix (LCP) Matrix

## 📜 Problem Statement
Given an `n x n` matrix `lcp`, where `lcp[i][j]` represents the **length of the longest common prefix** of the suffixes starting at indices `i` and `j`, reconstruct a valid string that satisfies this matrix.  
If no such string exists, return an empty string `""`.

## 🔍 Example Input/Output

**Example**

Input:
```

lcp = [
[3,0,1],
[0,2,0],
[1,0,1]
]

```

Output:
```

"aba"

```

Explanation:
- The constructed string `"aba"` satisfies all LCP conditions.

---

## 🧠 Approach

### 1. Greedy Character Assignment
- Initialize an empty result array `word`.
- Assign characters starting from `'a'` to `'z'`.

For each index `i`:
- If not assigned:
  - Assign a new character.
  - For all `j > i`, if:
```

lcp[i][j] > 0

```
  → assign same character to `word[j]`.

### 2. Character Limit Check
- If we exceed `'z'` (26 characters), return `""`.

### 3. Validate the Constructed String
Traverse all `(i, j)` pairs:

#### Case 1: Characters differ
```

word[i] != word[j]

```
Then:
```

lcp[i][j] must be 0

```

#### Case 2: Characters match
- If at boundary:
```

lcp[i][j] == 1

```

- Otherwise:
```

lcp[i][j] == lcp[i+1][j+1] + 1

````

### 4. Return Result
- If all validations pass → return the string
- Otherwise → return `""`

---

## 💻 Code 

```python
class Solution:
    def findTheString(self, lcp: List[List[int]]) -> str:
        n = len(lcp)
        word = [""] * n
        current = ord("a")

        for i in range(n):
            if not word[i]:
                if current > ord("z"):
                    return ""
                word[i] = chr(current)
                for j in range(i + 1, n):
                    if lcp[i][j]:
                        word[j] = word[i]
                current += 1

        for i in range(n - 1, -1, -1):
            for j in range(n - 1, -1, -1):
                if word[i] != word[j]:
                    if lcp[i][j]:
                        return ""
                else:
                    if i == n - 1 or j == n - 1:
                        if lcp[i][j] != 1:
                            return ""
                    else:
                        if lcp[i][j] != lcp[i + 1][j + 1] + 1:
                            return ""

        return "".join(word)
````

## ⏱ Complexity Analysis

**Time Complexity**

```
O(n^2)
```

* Constructing string: `O(n^2)`
* Validation: `O(n^2)`

**Space Complexity**

```
O(n)
```

* Result string storage.

## 🧪 Edge Cases

1. **Invalid LCP matrix**

```
Contradictions → return ""
```

2. **All zeros except diagonal**

```
Each character must be unique
```

3. **All values large**

```
String with repeated characters
```

4. **Exceeds 26 characters**

```
Return ""
```

5. **Single character**

```
lcp = [[1]] → return "a"
```
