1. Maintain a `stack` representing the currently active increasing thresholds.
2. Iterate over each number `n` in `nums`:
   - While the stackâ€™s top is greater than `n`, pop elements to maintain a non-decreasing order.
   - Skip if `n` is zero (it cannot create a new threshold).
   - If the stack is empty or its top is less than `n`, increment the operation counter and push `n`.
3. The final result `res` represents the number of new increasing levels introduced in the array.

```python
class Solution:
    def minOperations(self, nums: list[int]) -> int:
        stack = []
        res = 0
        for n in nums:
            while stack and stack[-1] > n:
                stack.pop()
            if n == 0:
                continue
            if not stack or stack[-1] < n:
                res += 1
                stack.append(n)
        return res
````
