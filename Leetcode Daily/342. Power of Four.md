We use **bit manipulation** to check:
1. `n > 0` â†’ must be positive.
2. `(n & (n - 1)) == 0` â†’ ensures `n` is a power of two (only one bit set).
3. `(n & 0x55555555) != 0` â†’ ensures that the set bit is at an **even position**, which is true only for powers of four.
   - `0x55555555` in binary is `010101010101...`, masking bits in even positions.

class Solution:
    def isPowerOfFour(self, n: int) -> bool:
        return n > 0 and (n & (n - 1)) == 0 and (n & 0x55555555) != 0
