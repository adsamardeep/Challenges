# 🏷 Mirror Distance of an Integer

## 📜 Problem Statement  
This function computes the absolute difference between a number and its digit-reversed (mirror) version. The mirror of a number is formed by reversing its digits. The goal is to return the distance between the original number and its mirrored value.

## 🔍 Example Input/Output  
Input:  
n = 123  

Output:  
198  

Explanation:  
Reversed number = 321  
Distance = |321 - 123| = 198  

## 🧠 Approach  
1. Initialize `rev = 0` to store the reversed number.  
2. Use a loop to extract digits from the original number:
   - Use `divmod(x, 10)` to get quotient and remainder.  
   - Append the remainder to `rev` by multiplying `rev` by 10 and adding the digit.  
3. Continue until all digits are processed.  
4. Compute the absolute difference between the reversed number and the original number.  
5. Return the result.

## 💻 Code 
```python
class Solution:
    def mirrorDistance(self, n: int) -> int:
        rev, x=0, n
        while x>0:
            x, r=divmod(x, 10)
            rev=10*rev+r
        return abs(rev-n)
````

## ⏱ Complexity Analysis

Time Complexity: O(d), where `d` is the number of digits in `n`.
Space Complexity: O(1), constant extra space used.

## 🧪 Edge Cases

* Single digit number → reversed is same → distance = 0.
* Number ending with zero (e.g., 120 → 21) → handled correctly.
* Palindrome numbers (e.g., 121) → distance = 0.
* Large numbers → still efficient due to digit-wise processing.
* n = 0 → loop doesn't run, result = 0.
