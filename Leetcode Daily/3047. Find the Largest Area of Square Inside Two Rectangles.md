You are given two lists:
- `bl`: bottom-left coordinates of rectangles
- `tr`: top-right coordinates of rectangles  

Each rectangle is axis-aligned.  
Your task is to find the **largest possible square area** that can be formed from the **overlapping region of any two rectangles**.

If no overlap exists that can form a square, return `0`.

## ðŸ§  Approach

### Key Insight

For **each pair of rectangles**:

1. Compute overlap boundaries:

   * `minX = max(left edges)`
   * `maxX = min(right edges)`
   * `minY = max(bottom edges)`
   * `maxY = min(top edges)`
2. If valid overlap exists:

   * Overlap width = `maxX - minX`
   * Overlap height = `maxY - minY`
3. The **largest square** inside the overlap has side:

   ```
   min(width, height)
   ```
4. Track the maximum square side across all pairs
5. Return sideÂ²

## ðŸ’» Code 

```python
class Solution:
    def largestSquareArea(self, bl: List[List[int]], tr: List[List[int]]) -> int:
        s = 0
        n = len(bl)

        for i in range(n):
            for j in range(i + 1, n):
                minX = max(bl[i][0], bl[j][0])
                maxX = min(tr[i][0], tr[j][0])
                minY = max(bl[i][1], bl[j][1])
                maxY = min(tr[i][1], tr[j][1])

                if minX < maxX and minY < maxY:
                    length = min(maxX - minX, maxY - minY)
                    s = max(s, length)

        return s * s
```
