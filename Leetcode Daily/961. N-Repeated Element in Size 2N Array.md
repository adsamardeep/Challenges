You are given an array of size `2N` containing `N + 1` unique elements, where exactly one element is repeated `N` times. The task is to identify and return the element that is repeated.

## ğŸ§  Approach

1. Iterate through the array up to the third-to-last element.
2. For each index `i`, compare `A[i]` with:

   * `A[i + 1]`
   * `A[i + 2]`
3. If either comparison matches, `A[i]` must be the element repeated `N` times.
4. If no match is found during the loop, the repeated element must be the **last element** of the array.

This works because in an array of size `2N` with one element repeated `N` times, that element must appear close together somewhere.

## ğŸ’» Code 

```python
class Solution:
    def repeatedNTimes(self, A: list[int]) -> int:
        for i in range(len(A) - 2):
            if A[i] == A[i + 1] or A[i] == A[i + 2]:
                return A[i]
        return A[-1]
```
