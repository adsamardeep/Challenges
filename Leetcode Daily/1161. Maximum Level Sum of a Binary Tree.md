## ğŸ“œ Problem Statement
Given the root of a binary tree, determine which level (1-indexed) has the **maximum sum of node values**. If multiple levels have the same maximum sum, return the **smallest level number**.

## ğŸ” Example Input/Output
**Input:**
```python
# Tree:
#       1
#      / \
#     7   0
#    / \
#   7  -8
````

**Output:**

```python
2
```

**Explanation:**
Level 2 has values `[7, 0]` with sum `7`, which is the maximum among all levels.

## ğŸ§  Approach

1. Perform a **level-order traversal (BFS)** using a queue.
2. For each level:

   * Compute the sum of node values at that level.
   * Track the level index and the maximum sum seen so far.
3. Update the answer whenever a higher sum is found.
4. Return the level index with the maximum sum.

## ğŸ’» Code 

```python
from typing import Optional
from collections import deque

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution:
    def maxLevelSum(self, root: Optional[TreeNode]) -> int:
        idx, Sum = 0, -inf
        q = deque()
        q.append(root)
        level = 1

        while q:
            qz = len(q)
            curSum = 0
            for i in range(qz):
                Node = q.popleft()
                curSum += Node.val
                if Node.left:
                    q.append(Node.left)
                if Node.right:
                    q.append(Node.right)

            if curSum > Sum:
                idx, Sum = level, curSum
            level += 1

        return idx
```

## â± Complexity Analysis

* **Time Complexity:** `O(n)`
  Each node is visited exactly once.
* **Space Complexity:** `O(w)`
  Where `w` is the maximum width of the tree (queue size).

## ğŸ§ª Edge Cases

* Single-node tree
* Tree with negative values only
* Multiple levels having the same sum (earliest level is chosen)
* Highly skewed tree (linked-list-like)
