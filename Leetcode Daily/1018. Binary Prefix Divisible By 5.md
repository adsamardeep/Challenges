# ğŸ§  Approach  
1. Maintain a running value `val` of the binary prefix modulo 5.  
2. For each bit `b`:
   - Shift left (`val << 1`) to make space for the new bit.
   - Add the bit.
   - Take modulo 5 to keep the number small.
3. If `val == 0`, the prefix is divisible by 5.  
4. Store the boolean result back into `nums` to reuse the same list.

```python
class Solution:
    def prefixesDivBy5(self, nums: List[int]) -> List[bool]:
        val = 0
        for i in range(len(nums)):
            val = ((val << 1) + nums[i]) % 5
            nums[i] = val == 0
        return nums
````
