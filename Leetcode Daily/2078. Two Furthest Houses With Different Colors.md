# 🏷 Two Furthest Houses With Different Colors

## 📜 Problem Statement  
You are given an array `colors` where `colors[i]` represents the color of the `i-th` house in a row. The task is to find the maximum distance between two houses that have different colors.

## 🔍 Example Input/Output  
Input:  
colors = [1, 1, 1, 6, 1, 1, 1]  

Output:  
3  

Explanation:  
The houses at indices `0` and `3` have different colors, and their distance is `|3 - 0| = 3`.  
No pair with different colors has a greater distance.

## 🧠 Approach  
1. Let `n` be the number of houses.  
2. Since we need the **maximum** distance, one of the houses should ideally be at one end of the array.  
3. Check from the right end backward to find the farthest house different from `colors[0]`.  
4. Check from the left end forward to find the farthest house different from `colors[n - 1]`.  
5. The answer is the maximum of these two distances.  
6. This works because the furthest valid pair must involve either the first house or the last house.

## 💻 Code 
```python
from typing import List

class Solution:
    def maxDistance(self, colors: List[int]) -> int:
        n = len(colors)
        ans = 0

        for i in range(n - 1, -1, -1):
            if colors[i] != colors[0]:
                ans = max(ans, i)
                break

        for i in range(n):
            if colors[i] != colors[n - 1]:
                ans = max(ans, n - 1 - i)
                break

        return ans
````

## ⏱ Complexity Analysis

Time Complexity: O(n), where `n` is the length of the array.
Space Complexity: O(1), constant extra space.

## 🧪 Edge Cases

* Only two houses with different colors → answer is 1.
* First and last house already have different colors → answer is `n - 1`.
* Many repeated colors in the middle → still works because only the ends matter.
* Minimum valid input size (`n = 2`) → handled correctly.
* The problem guarantees at least one valid pair with different colors.
