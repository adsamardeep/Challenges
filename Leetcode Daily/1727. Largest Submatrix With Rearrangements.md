## 🏷 Largest Submatrix with Rearrangements (Maximum Area of 1s)

## 📜 Problem Statement
Given a binary matrix, you can rearrange the columns of each row independently. The goal is to find the **largest possible submatrix consisting only of 1s** after rearranging columns optimally. Return the **maximum area** of such a submatrix.

## 🔍 Example Input/Output

**Example**

Input:
```
matrix = [
 [0,0,1],
 [1,1,1],
 [1,0,1]
]
```

Output:
```
4
```

Explanation:
After rearranging columns per row, we can form a submatrix of size `2 x 2` with all 1s.

---

## 🧠 Approach

### 1. Convert to Heights (Histogram Idea)
- Treat each column as a histogram.
- For each cell:
  - If it's `1`, accumulate height from the previous row.
  - If it's `0`, reset to `0`.

After this step, each row represents heights of consecutive `1`s ending at that row.

### 2. Sort Each Row
- Since column rearrangement is allowed, sort each row in **descending order**.
- This ensures taller columns are grouped together to maximize area.

### 3. Compute Maximum Area
For each row:
- For each column index `j`:
  - Height = `matrix[i][j]`
  - Width = `j + 1`
  - Area = `height * width`
- Track the maximum area.

### 4. Final Answer
Return the maximum area found across all rows.

---

## 💻 Code 

```python
class Solution:
    def largestSubmatrix(self, matrix: List[List[int]]) -> int:
        m = len(matrix)
        n = len(matrix[0])
        res = 0

        for i in range(1, m):
            for j in range(n):
                if matrix[i][j] == 1:
                    matrix[i][j] += matrix[i - 1][j]

        for i in range(m):
            matrix[i].sort(reverse=True)
            for j in range(n):
                res = max(res, matrix[i][j] * (j + 1))

        return res
```

---

## ⏱ Complexity Analysis

**Time Complexity**
```
O(m * n + m * n log n)
```
- Building heights: `O(m * n)`
- Sorting each row: `O(m * n log n)`

**Space Complexity**
```
O(1)
```
- In-place modification of the matrix.

## 🧪 Edge Cases

1. **All zeros**
```
matrix = [[0,0],[0,0]]
```
Output → `0`

2. **All ones**
```
matrix = [[1,1],[1,1]]
```
Output → `4`

3. **Single row**
```
matrix = [[1,0,1,1]]
```

4. **Single column**
```
matrix = [[1],[1],[0],[1]]
```

5. **Mixed values**
Handles arbitrary distributions of 0s and 1s efficiently.
