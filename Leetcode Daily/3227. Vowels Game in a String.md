# Alice Winning Condition (Check for Vowels in String)

📜 **Problem Statement**  
Alice and Bob play a game on a given string `s`. Alice wins if the string contains **at least one vowel** (`a, e, i, o, u`, in either upper or lowercase). Given a string `s`, determine if Alice wins.

🔍 **Example Input/Output**  
**Input**  
    s = "hello"

**Output**  
    True

**Explanation**  
The string contains vowels (`e`, `o`), so Alice wins.

**Input**  
    s = "rhythm"

**Output**  
    False

**Explanation**  
No vowels are present, so Alice loses.

🧠 **Approach (step-by-step)**  
1. For each character in the string `s`, check whether it is a vowel.  
2. The given code uses a **bitmask trick** with the number `0x208222` to quickly test whether a character (via its ASCII code modulo 32) corresponds to a vowel.  
   - `(ord(c) & 31)` maps characters to a range representing letters `A-Z` and `a-z`.  
   - The bitmask `0x208222` has `1`s in positions corresponding to vowels.  
   - Shifting and masking checks whether the character is a vowel.  
3. If any character in `s` is a vowel, return `True`. Otherwise, return `False`.

💻 **Code**  
    class Solution:
        def doesAliceWin(self, s: str) -> bool:
            # Check if any character in s is a vowel using bitmask trick
            return any(((0x208222 >> (ord(c) & 31)) & 1) for c in s)

⏱ **Complexity Analysis**  
- Let `n = len(s)`.  
- **Time:** O(n), since each character is checked once.  
- **Space:** O(1), only constant extra space for bitmask operations.

🧪 **Edge Cases**  
- Empty string → returns `False` (no vowels).  
- String with one vowel character (e.g., `"a"`) → returns `True`.  
- String with only consonants → returns `False`.  
- Mixed-case input (e.g., `"ApPlE"`) → works because bitmask trick handles both uppercase and lowercase.  
- Very long strings → still efficient because it’s O(n) with O(1) checks per character.
