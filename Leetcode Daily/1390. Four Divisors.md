## ðŸ§  Approach

1. A number has **exactly four divisors** only if it is:

   * The cube of a prime number: `pÂ³` â†’ divisors `{1, p, pÂ², pÂ³}`, or
   * The product of two **distinct primes**: `p Ã— q` â†’ divisors `{1, p, q, pÃ—q}`
2. For each number:

   * First, check if it is a perfect cube of a prime.
   * Otherwise, try to find a divisor pair `(a, b)` such that:

     * `a â‰  b`
     * Both `a` and `b` are prime.
3. If either condition is satisfied, compute and return the sum of its four divisors.
4. Accumulate results across all numbers.

## ðŸ’» Code 

```python
class Solution:
    def sumFourDivisors(self, nums: list[int]) -> int:
        res = 0
        for n in nums:
            val = self.sumOne(n)
            if val != -1:
                res += val
        return res

    def sumOne(self, n: int) -> int:
        p = round(n ** (1/3))
        if p ** 3 == n and self.isPrime(p):
            return 1 + p + p * p + p * p * p

        for i in range(2, int(n ** 0.5) + 1):
            if n % i == 0:
                a, b = i, n // i
                if a != b and self.isPrime(a) and self.isPrime(b):
                    return 1 + a + b + n
                return -1
        return -1

    def isPrime(self, x: int) -> bool:
        if x < 2:
            return False
        for i in range(2, int(x ** 0.5) + 1):
            if x % i == 0:
                return False
        return True
```
