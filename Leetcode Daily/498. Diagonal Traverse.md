# üü© Problem: Diagonal Traverse

## üìú Problem Statement
Given an `m x n` matrix, return all elements of the matrix in **diagonal order**.

---

## üîç Example 1
**Input**
    matrix = [
      [ 1, 2, 3 ],
      [ 4, 5, 6 ],
      [ 7, 8, 9 ]
    ]

**Output**
    [1,2,4,7,5,3,6,8,9]

---

## üîç Example 2
**Input**
    matrix = [[1,2],[3,4]]

**Output**
    [1,2,3,4]

---

## üß† Approach
We simulate the diagonal traversal:
1. Start at `(0,0)` and visit `m * n` elements.
2. Movement alternates by diagonals:
   - If `(row + col)` is even ‚Üí move **up-right**.
   - If `(row + col)` is odd ‚Üí move **down-left**.
3. Handle boundary conditions:
   - If moving up-right but at last column, go **down**.
   - If moving up-right but at first row, go **right**.
   - If moving down-left but at last row, go **right**.
   - If moving down-left but at first column, go **down**.

---

## üíª Code Implementation
    from typing import List

    class Solution:
        def findDiagonalOrder(self, matrix: List[List[int]]) -> List[int]:
            if not matrix or not matrix[0]:
                return []

            m, n = len(matrix), len(matrix[0])
            result = []
            row = col = 0

            for _ in range(m * n):
                result.append(matrix[row][col])

                if (row + col) % 2 == 0:  # moving up-right
                    if col == n - 1:
                        row += 1
                    elif row == 0:
                        col += 1
                    else:
                        row -= 1
                        col += 1
                else:  # moving down-left
                    if row == m - 1:
                        col += 1
                    elif col == 0:
                        row += 1
                    else:
                        row += 1
                        col -= 1

            return result

---

## ‚è± Complexity Analysis
- **Time Complexity:** O(m * n) ‚Äî we visit each element exactly once.
- **Space Complexity:** O(1) ‚Äî apart from the output array.

---

## üß™ Edge Cases
- Empty matrix ‚Üí return `[]`.
- Single row matrix ‚Üí output is that row.
- Single column matrix ‚Üí output is that column.
