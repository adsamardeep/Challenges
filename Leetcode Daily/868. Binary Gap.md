# 📏 Binary Gap

## 📜 Problem Statement

Given a positive integer `n`, return the **maximum distance between two consecutive `1`s** in its binary representation.

The distance between two `1`s is the difference between their bit positions.

If there is fewer than two `1`s, return `0`.

---

## 🔍 Example

### Example 1:
**Input:**  
n = 22  

**Binary:**  
10110  

Positions of `1`s → 0, 2, 4  
Distances → 2, 2  

**Output:**  
2  

---

### Example 2:
**Input:**  
n = 8  

**Binary:**  
1000  

Only one `1` →  

**Output:**  
0  

---

## 🧠 Key Idea

We scan through the bits and measure distance between consecutive `1`s.

---

## 🚀 Step-by-Step Explanation

### 1️⃣ Remove Trailing Zeros

```python
n //= n & -n
````

* `n & -n` isolates the **rightmost set bit**
* Dividing removes all trailing zeros
* This avoids unnecessary counting at the start

---

### 2️⃣ If Only One Bit Remains

```python
if n == 1: return 0
```

That means only one `1` exists.

---

### 3️⃣ Traverse Bits

We track:

* `gap` → distance since last `1`
* `max_gap` → maximum gap found

Logic:

* If current bit is `1`:

  * Update `max_gap`
  * Reset `gap`
* If current bit is `0`:

  * Increase `gap`

---

### 4️⃣ Why Return `max_gap + 1`?

Because:

* We counted zeros **between** ones
* Distance includes both endpoints

So:

```
distance = zeros_between + 1
```

---

## 💻 Code 

```python
class Solution:
    def binaryGap(self, n: int) -> int:
        n //= n & -n
        if n == 1: 
            return 0

        max_gap = 0
        gap = 0

        while n:
            if n & 1:
                max_gap = max(max_gap, gap)
                gap = 0
            else:
                gap += 1
            n >>= 1

        return max_gap + 1
```

---

## ⏱ Complexity Analysis

### Time Complexity:

* Traverse all bits
* O(log n)

### Space Complexity:

* Constant variables
* O(1)

---

## 🧪 Edge Cases

1. `n = 1` → 0
2. `n = 2 (10)` → 0
3. `n = 5 (101)` → 2
4. `n = 9 (1001)` → 3
5. Power of 2 numbers → 0

---

## 🎯 Why This Is Efficient

* Removes trailing zeros instantly
* Single pass through bits
* No string conversion
* Pure bit manipulation
