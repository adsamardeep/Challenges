# ⭐ Special Positions in a Binary Matrix

## 📜 Problem Statement

Given an `m x n` binary matrix `mat`,  
return the number of **special positions**.

A position `(i, j)` is special if:

- `mat[i][j] == 1`
- All other elements in row `i` are `0`
- All other elements in column `j` are `0`

---

## 🔍 Example

### Example 1:
**Input:**
```

mat = [
[1,0,0],
[0,0,1],
[1,0,0]
]

```

### Explanation:

Valid special position:
- `(1,2)` → only `1` in its row and column

**Output:**
```

1

````

---

## 🧠 Key Idea Behind This Bitmask Solution

Instead of scanning row and column repeatedly,  
this solution uses:

- Bitmasking for row checks
- Bit trick to detect single-bit rows
- Transpose to check columns

---

## 🚀 Step-by-Step Explanation

### 1️⃣ Convert Each Row into Bitmask

```python
def check1(row):
    ans = 0
    for i, x in enumerate(row):
        if x == 1:
            ans |= (1 << i)
    return ans
````

This converts:

```
[0,0,1,0]
```

into:

```
0100 (binary mask)
```

---

### 2️⃣ Check If Row Has Exactly One `1`

```python
single = mask and (mask & (mask - 1)) == 0
```

This classic trick checks if `mask` is a **power of 2**, meaning exactly one bit set.

---

### 3️⃣ Get Column Index of That Bit

```python
def ctz(x):
    return (x & -x).bit_length() - 1
```

* `x & -x` isolates rightmost set bit
* `.bit_length() - 1` gives its index

---

### 4️⃣ Column Validation

If row has single `1`:

* Check if that column hasn't already been used
* Count how many `1`s exist in that column

```python
cnt = sum(x for x in transpose[j])
ans += cnt == 1
```

Only if column also has exactly one `1`,
we increment answer.

---

### 5️⃣ Track Used Columns

```python
col |= (1 << j)
```

Prevents duplicate counting.

---

## 💻 Code 

```python
from typing import List

class Solution:
    def numSpecial(self, mat: List[List[int]]) -> int:
        def check1(row):
            ans = 0
            for i, x in enumerate(row):
                if x == 1:
                    ans |= (1 << i)
            return ans

        def ctz(x):
            return (x & -x).bit_length() - 1

        m, n = len(mat), len(mat[0])
        ans = 0
        col = 0
        transpose = list(zip(*mat))

        for row in mat:
            mask = check1(row)
            single = mask and (mask & (mask - 1)) == 0
            j = ctz(mask) if mask else -1

            if single and ((col >> j) & 1) == 0:
                col |= (1 << j)
                cnt = sum(x for x in transpose[j])
                ans += cnt == 1
            else:
                col |= mask

        return ans
```

---

## ⏱ Complexity Analysis

Let:

* `m` = rows
* `n` = columns

### Time Complexity:

* Row processing → O(mn)
* Column count checks → O(m)
* Overall → **O(mn)**

### Space Complexity:

* Transpose → O(mn)
* Bit masks → O(1)

---

## 🧪 Edge Cases

1. All zeros → 0
2. Single cell matrix → 1 if cell is 1
3. Multiple 1s in same row → not special
4. Multiple 1s in same column → not special
5. Large matrix

---

## 🎯 Why This Is Clever

* Uses bit tricks to detect single 1 in O(1)
* Avoids repeated full row scans
* Efficient and compact logic
