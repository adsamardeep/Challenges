## ğŸ§  Approach

This solution performs a **linear scan**:

1. Initialize the result as the first character (default wrap-around).
2. Track whether a valid greater character has been found.
3. When encountering a character greater than `target`:

   * If it's the first such character, store it.
   * Otherwise, keep the minimum among valid candidates.
4. Return the result.

Although correct, this approach does **not fully leverage the sorted property** of the input.

class Solution:
    def nextGreatestLetter(self, letters: List[str], target: str) -> str:
        res = letters[0]
        flag = False

        for ch in letters:
            if not flag:
                if ch > target:
                    res = ch
                    flag = not flag
            else:
                if ch > target and ch < res:
                    res = ch

        return res
```
