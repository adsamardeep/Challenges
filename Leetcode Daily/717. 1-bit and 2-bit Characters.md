You are given a binary array representing characters encoded with either 1-bit (`0`) or 2-bit (`10` or `11`) patterns. Determine whether the last character in the array is a 1-bit character.

# ðŸ” Example Input/Output
Input:
```

bits = [1, 0, 0]

```
Output:
```

True

````
Explanation: The encoding is `[10][0]`, so the last character is a 1-bit character.

# ðŸ§  Approach
1. Iterate through the array using index `i`.
2. If `bits[i] == 1`, skip two positions (2-bit character).  
   Otherwise skip one (1-bit character).
3. Stop when you reach the last index.
4. Return whether the pointer ends exactly on the last bit, meaning it's a 1-bit character.

# ðŸ’» Code 
```python
class Solution:
    def isOneBitCharacter(self, bits: List[int]) -> bool:
        n = len(bits)
        i = 0
        while i < n - 1:
            i += bits[i] + 1
        return i == n - 1
````
