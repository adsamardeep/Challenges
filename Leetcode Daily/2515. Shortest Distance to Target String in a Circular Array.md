# 🏷 Closest Target in Circular Array

## 📜 Problem Statement  
The function finds the minimum number of steps required to reach a target word in a circular list of words, starting from a given index. You can move either forward or backward, and the array wraps around. The goal is to return the shortest distance to any occurrence of the target word, or `-1` if it does not exist.

## 🔍 Example Input/Output  
Input:  
words = ["hello", "i", "am", "hello", "world"]  
target = "hello"  
s = 1  

Output:  
1  

Explanation:  
From index `1`, moving backward reaches index `0` ("hello") in 1 step.

## 🧠 Approach  
1. Let `n` be the length of the words array.  
2. Iterate `i` from `0` to `n//2`:
   - At each step, check both directions:
     - Forward: `(s + i) % n`
     - Backward: `(s - i) % n`
3. If either position contains the target word, return `i` as the minimum distance.  
4. If no match is found after checking all possible distances, return `-1`.  
5. Using bitwise OR `|` ensures both conditions are evaluated efficiently.

## 💻 Code 
```python
class Solution:
    def closestTarget(self, words: List[str], target: str, s: int) -> int:
        n = len(words)
        for i in range((n >> 1) + 1):
            if ((words[(s + i) % n] == target) |
                (words[(s - i) % n] == target)):
                return i
        return -1
````

## ⏱ Complexity Analysis

Time Complexity: O(n), in the worst case we check up to half the array.
Space Complexity: O(1), no extra space used.

## 🧪 Edge Cases

* Target at starting index → returns 0.
* Target not present → returns -1.
* Single element array → returns 0 if match, else -1.
* Multiple occurrences → returns the closest one.
* Even-sized array → both directions handled correctly due to modulo wrapping.
