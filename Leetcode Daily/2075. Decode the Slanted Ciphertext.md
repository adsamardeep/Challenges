🏷 Diagonal Ciphertext Decoder

📜 Problem Statement  
The function decodes a ciphertext string that was encoded by writing characters row-wise into a matrix and then reading them diagonally. Given the encoded string and the number of rows, the goal is to reconstruct the original plaintext by traversing diagonals starting from each column. Trailing spaces in the decoded result should be removed.

🔍 Example Input/Output  
Input:  
s = "ch   ie   pr", rows = 3  

Output:  
"cipher"  

🧠 Approach  
1. If the input string is empty, return it immediately.  
2. Compute the number of columns using ceiling division: `cols = ceil(len(s) / rows)`.  
3. Treat the string as a flattened matrix of size `rows x cols`.  
4. Iterate over each column as a starting point.  
5. For each starting column, traverse diagonally (increment both row and column).  
6. Append characters using index mapping: `s[r * cols + c]`.  
7. Continue until the diagonal goes out of bounds.  
8. Join all collected characters into a string.  
9. Remove trailing spaces using `.rstrip()`.

💻 Code 
```python
import math

class Solution:
    def decodeCiphertext(self, s: str, rows: int) -> str:
        if not s:
            return s

        cols = math.ceil(len(s) / rows)
        res = []

        for start in range(cols):
            r, c = 0, start
            while r < rows and c < cols:
                res.append(s[r * cols + c])
                r += 1
                c += 1

        return ''.join(res).rstrip()
````

⏱ Complexity Analysis
Time Complexity: O(n), where n is the length of the string (each character is visited once).
Space Complexity: O(n), for storing the result.

🧪 Edge Cases

* Empty string → returns empty string.
* rows = 1 → output is original string with trailing spaces removed.
* String with only spaces → returns empty string after `.rstrip()`.
* Perfect rectangle (no padding spaces) → works normally.
* Non-perfect rectangle → extra cells treated as spaces and trimmed at the end.
