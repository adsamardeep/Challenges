## 🏷 Check String Equality with Allowed Index Swaps

## 📜 Problem Statement
Given two strings `s1` and `s2` of length 4, determine if `s1` can be transformed into `s2` by performing any number of swaps between:
- indices `(0, 2)`
- indices `(1, 3)`

Return `True` if transformation is possible, otherwise `False`.

## 🔍 Example Input/Output

**Example**

Input:
```

s1 = "abcd"
s2 = "cbad"

```

Output:
```

True

```

Explanation:
- Swap indices `(0,2)` → `"cbad"`
- Swap indices `(1,3)` → `"cdab"` (optional, but not needed here)

---

## 🧠 Approach

### 1. Independent Swap Groups
Indices form two independent groups:
```

Group 1: (0, 2)
Group 2: (1, 3)

```
Each group can swap internally.

### 2. Check Group-wise Equality
For each group:
- Either characters match directly
- Or match after swapping

#### Group (0,2):
```

(s1[0], s1[2]) matches (s2[0], s2[2]) in any order

```

#### Group (1,3):
```

(s1[1], s1[3]) matches (s2[1], s2[3]) in any order

````

### 3. Final Condition
- Both groups must satisfy the condition.

---

## 💻 Code 

```python
class Solution:
    def canBeEqual(self, s1: str, s2: str) -> bool:
        return ((s1[0] == s2[0] and s1[2] == s2[2]) or
                (s1[0] == s2[2] and s1[2] == s2[0])) and \
               ((s1[1] == s2[1] and s1[3] == s2[3]) or
                (s1[1] == s2[3] and s1[3] == s2[1]))
````

## 🧪 Edge Cases

1. **Already equal strings**

```
s1 == s2 → True
```

2. **Only one group needs swapping**

```
"abcd" → "cbad"
```

3. **Impossible transformation**

```
"abcd" → "abdc"
```

4. **Repeated characters**

```
"aaaa" → "aaaa" → always True
```

5. **Minimum length constraint**

```
Strings always length = 4
```
