## ðŸ§  Approach
1. If `n` is less than or equal to 0, it cannot be a power of three â†’ return `False`.
2. While `n` is divisible by 3, keep dividing it by 3.
3. After the loop, if `n` equals 1 â†’ return `True`, else `False`.

## ðŸ’» Code Implementation
```python
class Solution:
    def isPowerOfThree(self, n: int) -> bool:
        if n <= 0:
            return False
        while n % 3 == 0:
            n //= 3
        return n == 1
```
