# 🔁 Minimum Changes To Make Alternating Binary String

## 📜 Problem Statement

Given a binary string `s`,  
return the **minimum number of operations** needed to make the string alternating.

An alternating string:
- Has no two adjacent characters equal
- Can be either:
  - `"010101..."`
  - `"101010..."`

Each operation consists of flipping a character (`0 → 1` or `1 → 0`).

---

## 🔍 Example

### Example 1
**Input:**
```

s = "0100"

```

Possible alternating strings:
```

0101  → 1 change
1010  → 3 changes

```

**Output:**
```

1

```

---

## 🧠 Key Insight

There are **only two valid alternating patterns**:

1. Pattern A → starts with `'0'`
```

index: 0 1 2 3 4 ...
char : 0 1 0 1 0 ...

```

2. Pattern B → starts with `'1'`
```

index: 0 1 2 3 4 ...
char : 1 0 1 0 1 ...

````

If we count mismatches with one pattern,
the mismatches with the other pattern = `n - count`.

So we only need to compute **one pattern's mismatch count**.

---

## 🚀 Why This XOR Trick Works

```python
count += (ord(s[i]) ^ i) & 1
````

### Explanation:

* `ord('0') = 48` (even)
* `ord('1') = 49` (odd)

So:

* `'0'` behaves like even
* `'1'` behaves like odd

If index `i` is even:

* expected char = `'0'`
  If index `i` is odd:
* expected char = `'1'`

Now:

```
(ord(s[i]) ^ i) & 1
```

* Returns `1` if parity differs (mismatch)
* Returns `0` if parity matches (correct)

So `count` = mismatches if pattern starts with `'0'`.

Other pattern mismatches = `n - count`.

We return:

```python
min(count, n - count)
```

---

## 💻 Code 

```python
class Solution:
    def minOperations(self, s: str) -> int:
        count, n = 0, len(s)
        for i in range(n):
            count += (ord(s[i]) ^ i) & 1
        
        return min(count, n - count)
```

---

## ⏱ Complexity Analysis

* Time Complexity → **O(n)**
* Space Complexity → **O(1)**

Only one pass through the string.

---

## 🧪 Edge Cases

1. `"0"` → 0
2. `"1"` → 0
3. Already alternating → 0
4. All same characters → n // 2
5. Large input (10⁵ length) → still O(n)

---

## 🎯 Why This Is Elegant

* Avoids building two pattern strings
* Uses bit manipulation trick
* One pass solution
* Constant space
