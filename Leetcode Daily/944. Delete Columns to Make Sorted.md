Given an array of equal-length strings `strs`, determine how many columns must be deleted so that the remaining columns make the array **lexicographically sorted** from top to bottom. A column must be deleted if it violates the non-decreasing order when read vertically.

## ðŸ” Example Input/Output
**Input:**

**Output:**


**Explanation:**

* Column 0: `c, d, g` â†’ sorted âœ”
* Column 1: `b, a, h` â†’ not sorted âœ˜ (delete)
* Column 2: `a, f, i` â†’ sorted âœ”

## ðŸ§  Approach

1. Iterate over each column index.
2. For a given column, compare characters row by row from top to bottom.
3. If any character is smaller than the one directly above it, the column violates lexicographic order.
4. Increment the deletion count and stop checking further rows for that column.
5. Continue for all columns and return the total count.

## ðŸ’» Code 
```python
from typing import List

class Solution:
    def minDeletionSize(self, strs: List[str]) -> int:
        res = 0

        for col in range(len(strs[0])):
            for row in range(1, len(strs)):
                if strs[row][col] < strs[row - 1][col]:
                    res += 1
                    break

        return res
```
