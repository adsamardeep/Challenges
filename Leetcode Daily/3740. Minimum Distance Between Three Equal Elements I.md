# 🏷 Minimum Distance Between Equal Elements

## 📜 Problem Statement  
This function finds the minimum distance between two equal values in the array. If a value appears more than once, the distance is calculated based on the difference between their positions, and the function returns the smallest such distance. If no duplicate element exists, it returns `-1`.

## 🔍 Example Input/Output  
Input:  
nums = [1, 2, 3, 1, 4]  

Output:  
6  

Explanation:  
The value `1` appears at positions `1` and `4` (1-based indexing).  
Distance = `(4 - 1) * 2 = 6`.

## 🧠 Approach  
1. Initialize an array `last2` to track the last two seen positions for each value.  
2. Traverse the array and use 1-based indexing for positions.  
3. For each number:
   - Decode its stored packed value into:
     - `old`: the second-last occurrence
     - `cur`: the last occurrence  
4. Update the packed record so the previous last occurrence becomes old, and the current position becomes the new last occurrence.  
5. If `old` exists, compute the distance `(pos - old) << 1`, which is equivalent to `2 * (pos - old)`.  
6. Keep track of the minimum such distance across all values.  
7. If no duplicate was found, return `-1`; otherwise return the minimum distance.

## 💻 Code 
```python
class Solution:
    def minimumDistance(self, nums: List[int]) -> int:
        n = len(nums)
        last2 = [0] * n
        res = 200

        for i in range(n):
            val, pos = nums[i] - 1, i + 1
            pack = last2[val]
            old, cur = pack & 255, pack >> 8

            last2[val] = cur | (pos << 8)

            if old:
                res = min(res, (pos - old) << 1)

        return -(res == 200) | res
````

## ⏱ Complexity Analysis

Time Complexity: O(n), where `n` is the length of the array because each element is processed once.
Space Complexity: O(n), due to the `last2` tracking array.

## 🧪 Edge Cases

* No duplicate elements → returns `-1`.
* Exactly one duplicate pair → returns that pair’s computed distance.
* Multiple duplicates of the same value → checks the smallest distance using consecutive tracked occurrences.
* All elements same → minimum distance comes from adjacent occurrences.
* Single element array → returns `-1`.
* The implementation assumes values are in the range `1` to `n`, since it uses `nums[i] - 1` as an index.
