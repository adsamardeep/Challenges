## 🏷 Check Matrix Similarity After Alternating Row Shifts

## 📜 Problem Statement
Given a matrix `mat` and an integer `k`, determine if the matrix remains the same after applying:
- **Left shift by `k`** on even-indexed rows
- **Right shift by `k`** on odd-indexed rows  

Return `True` if the matrix is unchanged after these operations, otherwise return `False`.

## 🔍 Example Input/Output

**Example**

Input:
```

mat = [
[1,2,3],
[4,5,6]
]
k = 3

```

Output:
```

True

```

Explanation:
- Since `k % n = 0`, no actual shift happens → matrix remains unchanged.

---

## 🧠 Approach

### 1. Normalize k
- Shifting by `n` (row length) results in the same row:
```

k = k % n

```

### 2. Simulate Shift Virtually
Instead of actually shifting rows:
- Compare each element with its expected shifted position.

### 3. Even Rows (Left Shift)
For row `i` even:
```

mat[i][j] == mat[i][(j + k) % n]

```

### 4. Odd Rows (Right Shift)
For row `i` odd:
```

mat[i][j] == mat[i][(j - k) % n]

````

### 5. Early Exit
- If any mismatch occurs → return `False`

### 6. Final Result
- If all elements match expected positions → return `True`

---

## 💻 Code 

```python
class Solution:
    def areSimilar(self, mat, k):
        m, n = len(mat), len(mat[0])
        
        k %= n  # (reduce k<n)
        
        for i in range(m):
            for j in range(n):
                if i % 2 == 0:
                    # even row , left shift
                    if mat[i][j] != mat[i][(j + k) % n]:
                        return False
                else:
                    # odd row , right shift
                    if mat[i][j] != mat[i][(j - k) % n]:
                        return False
        
        return True
````

---

## ⏱ Complexity Analysis

**Time Complexity**

```
O(m * n)
```

* Each element is checked once.

**Space Complexity**

```
O(1)
```

* No extra space used.

---

## 🧪 Edge Cases

1. **k = 0**

```
No shift → always True
```

2. **k ≥ n**

```
Handled by modulo reduction
```

3. **Single column**

```
n = 1 → always True
```

4. **Single row**

```
Only left shift applies
```

5. **Mismatch early**

```
Returns False immediately
```
