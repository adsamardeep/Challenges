ğŸ§  **Approach (step-by-step)**  
1. Iterate once through the string, grouping consecutive equal colors.  
2. For each group of equal colors, to avoid adjacent equal colors you must remove all but the one with the maximum removal time.  
3. As you scan, keep `curr_max_time` for the ongoing group and add `min(curr_max_time, neededTime[i])` to the answer when you encounter a same-color balloon â€” this effectively removes the cheaper of the two (keeping the larger).  
4. When color changes, reset the group state.

ğŸ’» **Code (single code block, indented)**  
    class Solution:
        def minCost(self, colors: str, neededTime: list[int]) -> int:
            total_time, curr_max_time = 0, 0

            for i in range(len(colors)):
                # New color group: reset the running max
                if i > 0 and colors[i] != colors[i - 1]:
                    curr_max_time = 0

                # Remove the cheaper balloon between current and the best so far in the group
                total_time += min(curr_max_time, neededTime[i])

                # Update the max time in this group
                curr_max_time = max(curr_max_time, neededTime[i])

            return total_time
