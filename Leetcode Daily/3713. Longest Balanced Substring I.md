> The number of distinct characters equals  
> The number of characters that appear the **maximum frequency** in that substring.

## ğŸ’¡ What â€œBalancedâ€ Means

For a substring:

- `uniq` = number of distinct letters
- `maxF` = highest frequency of any letter
- `cntMax` = how many letters appear `maxF` times

The substring is **balanced** if:


## ğŸ§  Intuition

For every possible starting index `l`, expand the substring to the right:

At each step:

1. Update frequency of current character
2. Track:
   - number of distinct characters (`uniq`)
   - current maximum frequency (`maxF`)
   - how many characters have that maximum frequency (`cntMax`)
3. If `uniq == cntMax`, update answer

This checks every substring â†’ brute-force with optimization.

## ğŸ’» Code 

```python
class Solution:
    def longestBalanced(self, s: str) -> int:
        cnt, n = 1, len(s)

        for l in range(n):
            freq = [0] * 26
            uniq, maxF, cntMax = 0, 0, 0

            for r in range(l, n):
                idx = ord(s[r]) - 97
                freq[idx] += 1
                f = freq[idx]

                if f == 1:
                    uniq += 1

                if f > maxF:
                    maxF = f
                    cntMax = 1
                elif f == maxF:
                    cntMax += 1

                if uniq == cntMax:
                    cnt = max(cnt, r - l + 1)

        return cnt
````
