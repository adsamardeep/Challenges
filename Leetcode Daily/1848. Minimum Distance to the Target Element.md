# 🏷 Minimum Distance to Target Element

## 📜 Problem Statement  
This function finds the minimum distance between a given `start` index and any occurrence of a specified `target` value in the array. The distance is measured as the absolute difference between indices. The goal is to return the smallest such distance.

## 🔍 Example Input/Output  
Input:  
nums = [1, 2, 3, 4, 5]  
target = 5  
start = 3  

Output:  
1  

Explanation:  
The target `5` is at index `4`, so distance = `|4 - 3| = 1`.

## 🧠 Approach  
1. Initialize a variable `mindist` with infinity to track the minimum distance.  
2. Iterate through the array using `enumerate` to get both index and value.  
3. For each element:
   - If it equals the target, compute the absolute difference between its index and `start`.  
   - Update `mindist` if the computed distance is smaller.  
4. After traversal, return the minimum distance found.

## 💻 Code 
```python
class Solution:
    def getMinDistance(self, nums: list[int], target: int, start: int) -> int:
        mindist = float('inf')
        for i, num in enumerate(nums):
            if num == target:
                mindist = min(mindist, abs(i - start))
        return mindist
````

## ⏱ Complexity Analysis

Time Complexity: O(n), where `n` is the length of the array.
Space Complexity: O(1), constant extra space.

## 🧪 Edge Cases

* Target appears multiple times → returns the closest occurrence.
* Target appears at `start` index → distance is 0.
* Target not present → function returns `inf` (depends on constraints, usually guaranteed to exist).
* Single element array → returns 0 if it matches target.
* Large array → still efficient due to single pass.
