## 🏷 Maximum Walls Destroyed by Robots (DP with Overlap Handling)

## 📜 Problem Statement
Given:
- `robots[i]` → position of robot `i`
- `distance[i]` → how far robot `i` can destroy walls
- `walls` → positions of walls  

Each robot can destroy walls either:
- **to the left** within its distance
- **to the right** within its distance  

Robots cannot double-count walls, and overlapping destruction must be handled carefully.  
Return the **maximum number of walls destroyed**.

---

## 🔍 Example Input/Output

**Example**

Input:
```

robots = [2, 6]
distance = [2, 2]
walls = [1,2,3,5,6,7]

```

Output:
```

5

```

Explanation:
- Robot 1 destroys [1,2]
- Robot 2 destroys [5,6,7]
- Total = 5

---

## 🧠 Approach

### 1. Sort Inputs
- Sort robots by position:
```

robots = sorted(zip(robots, distance))

```

- Sort walls for binary search.

---

### 2. Preprocessing
- Append a dummy robot at infinity to simplify right boundary handling.

---

### 3. Helper Function
Count walls in range `[l, r]`:
```

count = bisect_right(walls, r) - bisect_left(walls, l)

```

---

### 4. DP Definition
```

dp[i][0] → max walls if robot i goes LEFT
dp[i][1] → max walls if robot i goes RIGHT

```

---

### 5. Base Case (First Robot)
- Compute:
  - Left destruction range
  - Right destruction range (bounded by next robot)

---

### 6. Transition

For each robot `i`:

#### Right Move
```

dp[i][1] = max(dp[i-1][0], dp[i-1][1]) + right_gain

```

---

#### Left Move (No Overlap)
```

dp[i][0] = dp[i-1][0] + left_gain

```

---

#### Left Move (Overlap Case)
- If previous robot went right, overlap may occur:
```

overlap = walls counted in overlapping range

```

Adjust:
```

dp[i][0] = max(dp[i][0], dp[i-1][1] + left_gain - overlap)

```

---

### 7. Final Answer
```

max(dp[n-1][0], dp[n-1][1])

````

---

## 💻 Code 

```python
class Solution:
    def maxWalls(self, robots: List[int], distance: List[int], walls: List[int]) -> int:
        n = len(robots)

        robots = sorted(zip(robots, distance))
        walls.sort()

        robots.append((10**18, 0))

        def count_walls(l, r):
            if l > r:
                return 0
            return bisect.bisect_right(walls, r) - bisect.bisect_left(walls, l)

        dp = [[0, 0] for _ in range(n)]

        pos, dist = robots[0]

        left_gain = count_walls(pos - dist, pos)
        right_gain = count_walls(pos, min(robots[1][0] - 1, pos + dist))

        dp[0][0] = left_gain
        dp[0][1] = right_gain

        for i in range(1, n):
            pos, dist = robots[i]
            prev_pos, prev_dist = robots[i - 1]

            left_l = max(pos - dist, prev_pos + 1)
            left_r = pos
            left_gain = count_walls(left_l, left_r)

            right_l = pos
            right_r = min(robots[i + 1][0] - 1, pos + dist)
            right_gain = count_walls(right_l, right_r)

            dp[i][1] = max(dp[i - 1][0], dp[i - 1][1]) + right_gain

            dp[i][0] = dp[i - 1][0] + left_gain

            overlap_l = left_l
            overlap_r = min(prev_pos + prev_dist, pos - 1)
            overlap = count_walls(overlap_l, overlap_r)

            dp[i][0] = max(dp[i][0], dp[i - 1][1] + left_gain - overlap)

        return max(dp[n - 1][0], dp[n - 1][1])
````

## ⏱ Complexity Analysis

**Time Complexity**

```id="t1"
O(n log m)
```

* Sorting: `O(n log n)`
* Each DP step uses binary search on walls → `log m`

---

**Space Complexity**

```id="t2"
O(n)
```

* DP array

---

## 🧪 Edge Cases

1. **No walls**

```id="e1"
Return 0
```

2. **Single robot**

```id="e2"
Choose max(left, right)
```

3. **Overlapping ranges**

```id="e3"
Handled via overlap subtraction
```

4. **Robots very close**

```id="e4"
Left range constrained by previous robot
```

5. **Large coordinates**

```id="e5"
Handled efficiently via binary search
```
