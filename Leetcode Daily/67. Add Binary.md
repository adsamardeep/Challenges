# ➕ Add Binary (Without Using `+`)

## 📜 Problem Statement

Given two binary strings `a` and `b`, return their sum as a binary string.

You must simulate binary addition.

---

## 🧠 Key Insight

Binary addition can be done using **bitwise operations**:

- `x ^ y` → sum without carry  
- `x & y` → carry bits  
- `(x & y) << 1` → shifted carry  

We repeat until there is no carry left.

This mimics how hardware performs addition.

---

## 💻 Code 

```python
class Solution:
    def addBinary(self, a, b) -> str:
        x, y = int(a, 2), int(b, 2)
        while y:
            x, y = x ^ y, (x & y) << 1
        return bin(x)[2:]
````

---

## 🔎 Example

### Input

```python
a = "1010"
b = "1011"
```

### Step-by-step

```
x = 10
y = 11

Iteration 1:
x ^ y = 1
carry = (10 & 11) << 1 = 20

Iteration 2:
x ^ y = 21
carry = 0
```

Final result:

```
10101
```

---

## ⏱ Complexity

* **Time:** `O(n)`
  (number of bits)
* **Space:** `O(1)`
  (integer operations)

---

## 🧪 Edge Cases

* `"0" + "0"` → `"0"`
* Very large binary strings → Python handles big integers
* One string empty → treated as 0

---

## 🚀 Why This Works

Binary addition rule:

```
sum_bit = a XOR b
carry   = a AND b
```

We repeat until carry becomes zero.

This is exactly how digital circuits implement addition.
