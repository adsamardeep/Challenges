# ğŸ§  Approach
Let the length of the interval be `total = high - low + 1`.

- If both `low` and `high` are odd, then among these `total` numbers, the number of odd elements is `total//2 + 1`.
- Otherwise, the count is simply `total//2`.

This works because odd/even alternates. Only when both ends are odd does the interval contain one extra odd.

# ğŸ’» Code 
```python
class Solution:
    def countOdds(self, low: int, high: int) -> int:
        total = high - low + 1
        if low % 2 != 0 and high % 2 != 0:
            return total // 2 + 1
        return total // 2
````
