Given a binary array `nums` and an integer `k`, determine whether every pair of consecutive `1`s in the array is separated by at least `k` zeros. Return `True` if the spacing condition holds, otherwise `False`.

# ğŸ” Example Input/Output
Input:

```
Output:
```

False

````
Explanation: The last two `1`s are only 1 index apart (distance < 2).

# ğŸ§  Approach
1. Track the index of the last seen `1` using `prev`, initially `None`.
2. Iterate through `nums` with index `i`:
   - If `nums[i] == 1`, check if a previous `1` exists.
   - If so, verify whether `i - prev > k`. If not, the spacing is violated â†’ return `False`.
   - Update `prev = i`.
3. If the loop completes without conflict, return `True`.
4. If `k == 0`, return `True` immediately since any spacing is allowed.

# ğŸ’» Code 
```python
class Solution:
    def kLengthApart(self, nums: List[int], k: int) -> bool:
        if k == 0:
            return True
        prev = None
        for i, num in enumerate(nums):
            if num == 1:
                if prev is not None and i - prev <= k:
                    return False
                prev = i
        return True
````

* **Space Complexity:** O(1), using only constant extra variables.
