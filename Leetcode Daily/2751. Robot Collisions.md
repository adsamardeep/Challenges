## 🏷 Surviving Robots After Collisions

## 📜 Problem Statement
Given arrays `positions`, `healths`, and a string `directions` representing robots on a line:
- Each robot moves either **left ('L')** or **right ('R')**
- When two robots collide:
  - The robot with **lower health is destroyed**
  - The stronger robot loses **1 health**
  - If equal, both are destroyed  

Return the healths of robots that survive, in their original order.

---

## 🔍 Example Input/Output

**Example**

Input:
```

positions = [3,5,2,6]
healths = [10,10,15,12]
directions = "RLRL"

```

Output:
```

[14]

```

Explanation:
- Robots collide based on positions and directions.
- Only one robot survives with updated health.

---

## 🧠 Approach

### 1. Sort by Position
- Robots must be processed in **left-to-right order**:
```

order = sorted indices based on positions

```

---

### 2. Use Stack for Right-Moving Robots
- Maintain a stack of indices of robots moving right (`'R'`)
- These are potential collision candidates with incoming `'L'` robots

---

### 3. Process Each Robot

#### Case 1: दिशा = 'R'
- Push index into stack

#### Case 2: दिशा = 'L'
- Check collisions with stack top:
```

while stack:

```

Compare healths:

---

### 4. Collision Rules

#### Case A: Right robot weaker
```

h[top] < h[idx]

```
- Right robot dies
- Left robot loses 1 health
- Continue checking

---

#### Case B: Left robot weaker
```

h[top] > h[idx]

```
- Left robot dies
- Right robot loses 1 health
- Stop collision

---

#### Case C: Equal health
```

h[top] == h[idx]

````
- Both die
- Stop collision

---

### 5. Track Survivors
- Use `alive[]` array to mark surviving robots

---

### 6. Final Output
- Return healths of alive robots in original order

---

## 💻 Code 

```python
class Solution:
    def survivedRobotsHealths(self, positions, healths, directions):

        n = len(positions)
        order = sorted(range(n), key=lambda i: positions[i])

        h = healths[:]
        alive = [True]*n
        stack = []

        for idx in order:

            if directions[idx] == 'R':
                stack.append(idx)

            else:
                while stack:

                    top = stack[-1]

                    if h[top] < h[idx]:
                        alive[top] = False
                        stack.pop()
                        h[idx] -= 1

                    elif h[top] > h[idx]:
                        alive[idx] = False
                        h[top] -= 1
                        break

                    else:
                        alive[top] = False
                        alive[idx] = False
                        stack.pop()
                        break

        return [h[i] for i in range(n) if alive[i]]
````

## ⏱ Complexity Analysis

**Time Complexity**

```
O(n log n)
```

* Sorting dominates

**Space Complexity**

```
O(n)
```

* Stack and auxiliary arrays

## 🧪 Edge Cases

1. **No collisions**

```
All robots move same direction
```

2. **All robots collide**

```
Chain reactions handled via stack
```

3. **Equal health collisions**

```
Both robots destroyed
```

4. **Single robot**

```
Always survives
```

5. **Large inputs**
   Efficient due to single pass after sorting
