## 🏷 Generate String from Pattern Constraints (T/F Substring Matching)

## 📜 Problem Statement
Given two strings `s` and `t`, construct a string `ans` such that:
- For every index `i` where `s[i] == 'T'`, the substring `ans[i : i + len(t)]` **must equal** `t`
- For every index `i` where `s[i] == 'F'`, the substring `ans[i : i + len(t)]` **must NOT equal** `t`

Return any valid string `ans`, or an empty string `""` if it's impossible.

---

## 🔍 Example Input/Output

**Example**

Input:
```

s = "TFT"
t = "ab"

```

Output:
```

"abab"

```

Explanation:
- At `i = 0` → must match `"ab"`
- At `i = 1` → must NOT match `"ab"`
- At `i = 2` → must match `"ab"`

---

## 🧠 Approach

### 1. Initialize Result
- Length of result:
```

n + m - 1

```

- Fill with `'?'` (unknown values)

---

### 2. Enforce 'T' Constraints
For every `i` where:
```

s[i] == 'T'

```

- Force:
```

ans[i + j] = t[j]

```

- If conflict occurs → return `""`

---

### 3. Fill Remaining Positions
- Replace all `'?'` with `'a'` (default safe character)

---

### 4. Enforce 'F' Constraints
For every `i` where:
```

s[i] == 'F'

```

- If substring equals `t`, we must break it:
```

ans[i : i+m] != t

````

### Fix Strategy:
- Traverse from right to left in the window
- Find a position originally `'?'`
- Change `'a' → 'b'` to break equality

If no such position exists → return `""`

---

### 5. Return Result
- Join and return final string

---

## 💻 Code 

```python
class Solution:
    def generateString(self, s: str, t: str) -> str:
        n, m = len(s), len(t)
        ans = ['?'] * (n + m - 1)  # ? indicates a pending position
        
        # Process 'T'
        for i, b in enumerate(s):
            if b != 'T':
                continue
            # The substring must match t
            for j, c in enumerate(t):
                v = ans[i + j]
                if v != '?' and v != c:
                    return ""
                ans[i + j] = c
        
        old_ans = ans
        ans = ['a' if c == '?' else c for c in ans]  # Initial default is 'a'
        
        # Process 'F'
        for i, b in enumerate(s):
            if b != 'F':
                continue
            # Substring must not equal t
            if ''.join(ans[i: i + m]) != t:
                continue
            # Locate the last pending position to modify
            for j in range(i + m - 1, i - 1, -1):
                if old_ans[j] == '?':  # Change 'a' to 'b'
                    ans[j] = 'b'
                    break
            else:
                return ""
        return ''.join(ans)
````

---

## ⏱ Complexity Analysis

**Time Complexity**

```
O(n * m)
```

* Processing each constraint may scan up to `m` characters

**Space Complexity**

```
O(n + m)
```

* Result array storage

---

## 🧪 Edge Cases

1. **Conflicting 'T' constraints**

```
Overlap mismatch → return ""
```

2. **No '?' available for 'F' fix**

```
Cannot break equality → return ""
```

3. **All 'T'**

```
Fully determined string
```

4. **All 'F'**

```
Flexible construction with modifications
```

5. **t length = 1**

```
Simplifies to character constraints
```
